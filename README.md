Spotify Data Analysis & Visualization

Analyzed Spotify’s financial and user data to uncover trends in revenue, user growth, and profitability.  
Performed data cleaning and exploratory data analysis using Python.  
Built visualizations to compare Premium vs Ad-supported revenue streams.  
Identified consistent revenue growth and strong dominance of Premium subscriptions.  
Observed fluctuations and inefficiencies in ad-based profitability.  
Derived insights on user growth trends and stable Premium ARPU.  

**Tech Stack:** Python, Pandas, NumPy, Matplotlib, Seaborn
